
# CrissCross App

Mobile App Frontend

### Install

In `CrissCross_app/src/config.tsx` change googleSignInConfig to reflect your credentials. 

In `CrissCross_app/src/network/config.tsx` change domains to reflect your domains for development. 

Start the app

    npm install
    expo start

Use the expo client app to run
